# SS-02 Reference

## Decision Variable

- The key variable is the speaker's motive for departing from plain factual reporting.

## Route Signal

- Use this skill when the task asks why a nonliteral, selective, mistaken, polite, or self-protective statement was made.

## Hard Boundary

- If the task only asks whether the statement is true, route to `skill14`.
- If there is no statement motive to explain, do not use this skill.

## Shortcut To Avoid

- Do not label every mismatch as deliberate lying.
- Do not answer with hidden facts alone; explain why that wording was chosen.

## Common Failure Modes

- Collapsing forgetting, confusion, politeness, and deception into a single `lie` category.
- Explaining what happened but not why the speaker chose that particular wording.
- Confusing truth judgment with motive explanation.

## Minimal Pair

- Case A: the same mismatch can come from forgetting or confusion.
- Case B: the same mismatch can come from face-saving, self-protection, or conflict avoidance.

## Boundary Stress Test

- `Why did she say that?` -> this skill.
- `Was what she said true?` -> `skill14`.

## Generalization Note

- The transferable core is action explanation in communicative behavior: infer the speaker's local goal, pressure, or memory state that made this wording instrumentally useful.
