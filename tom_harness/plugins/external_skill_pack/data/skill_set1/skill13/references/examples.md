# PS-01 Reference

## Decision Variable

- The key variable is which persuasion lever best matches the listener's actual incentive or resistance.

## Route Signal

- Use this skill when the task asks for the most effective way to get another person to agree or act.

## Hard Boundary

- If the task is about decoding what an indirect sentence already means, route to `skill12`.
- If no strategy choice about influencing a listener is involved, do not use this skill.

## Shortcut To Avoid

- Do not center only the persuader's need.
- Do not give generic encouragement when the listener's barrier is specific.

## Common Failure Modes

- Offering a pleasant-sounding message that ignores the listener's real cost, fear, or competing desire.
- Confusing explanation of meaning with strategy for influence.
- Forgetting that different barriers call for different persuasion levers.

## Minimal Pair

- Case A: if the listener resists because of cost, compensation or convenience is the right lever.
- Case B: if the listener resists because of uncertainty, reassurance or evidence is stronger.

## Boundary Stress Test

- `What should she say to persuade him?` -> this skill.
- `What did she really mean by saying that?` -> `skill12`.

## Generalization Note

- The transferable core is audience modeling: effective persuasion depends on the listener's decision bottleneck, not generic positivity.
